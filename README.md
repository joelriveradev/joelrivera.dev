# joelrivera.dev

Personal developer site for Joel Rivera, AI Software Engineer.

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- shadcn/ui
- next-themes

## Development

```bash
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
bun run build
bun start
```
